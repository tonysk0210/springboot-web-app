課程詳細頁：載入課程資料與準備新增學生流程

當管理員點選 「View」 按鈕時，系統會透過 URL 參數中的 courseId， 由 AdminController 使用 @RequestParam("courseId") 取得對應的課程 ID， 並透過 coursesRepository.findById(courseId) 查詢目標 Course 物件。

查詢結果將加入 Model，傳送至前端頁面進行渲染。 前端會根據該 Course 物件的屬性顯示課程名稱， 並透過其 Set<Person> 屬性列出所有已註冊此課程的學生清單。

此外，系統也會提供一個新的 Person 物件，作為 「Add Student」 表單的綁定對象， 供管理員輸入欲加入此課程的學生 Email。

為了在後續新增學生的流程中能正確建立關聯，系統會將目前查詢到的 Course 物件 儲存至 HttpSession 中（屬性名稱為 "courseInSession"），以供後續處理使用。

課程學生新增邏輯說明與錯誤處理機制（Add Student）

由 addCourseStudent(Person person, HttpSession session) 處理新增學生邏輯：

@PostMapping("/addCourseStudent")
public String addCourseStudent(Person person, HttpSession session) {

    Courses courseInSession = (Courses) session.getAttribute("courseInSession"); // 從 Session 取得當前要新增學生的 Course 物件

    Person thePerson = personRepository.readByEmail(person.getEmail()); // 根據輸入的 email 查詢學生是否存在（驗證學生帳號）

    if (thePerson == null)
        return "redirect:/admin/viewCourseDetail?courseId=" + courseInSession.getCourseId() + "&error=personNotFound"; // 若學生不存在，重新導向回課程頁面並附帶 error=personNotFound，顯示「學生不存在」
    if (courseInSession.getPersons().contains(thePerson)) {
        return "redirect:/admin/viewCourseDetail?courseId=" + courseInSession.getCourseId() + "&error=alreadyEnrolled"; // 若該學生已註冊此課程，重新導向並附帶 error=alreadyEnrolled，顯示「學生已註冊」
    }

    thePerson.getCourses().add(courseInSession); // 由擁有端 Person 設定關聯（此為有效的資料操作端）
    courseInSession.getPersons().add(thePerson); // 同步更新被參照端以維持物件模型一致性（可選擇性加入）
    personRepository.save(thePerson); // 儲存變更，更新資料庫中的關聯資料

    return "redirect:/admin/viewCourseDetail?courseId=" + courseInSession.getCourseId(); // 新增成功後，重新導向至當前課程頁面並刷新學生清單
}

「學生不存在」　錯誤信息：　⚠️ Student not found
「學生已註冊」　錯誤信息：　⚠️ Student is already enrolled in this course

課程學生移除功能邏輯說明（Remove Student）

@GetMapping("/deleteCourseStudent")
public String deleteCourseStudent(@RequestParam int personId, HttpSession session) {
    Optional<Person> personTobeRemovedOptional = personRepository.findById(personId);
    Person personTobeRemoved = personTobeRemovedOptional.get(); // 根據「Remove」傳入的 personId 取得對應的 Person 物件

    Courses courseInSession = (Courses) session.getAttribute("courseInSession"); // 取得當前課程 Course 物件

    personTobeRemoved.getCourses().remove(courseInSession); // 由擁有端 Person 解除與 Course 的關聯
    personRepository.save(personTobeRemoved); //　儲存變更，更新資料庫中的關聯資訊
    return "redirect:/admin/viewCourseDetail?courseId=" + courseInSession.getCourseId();　// 關聯解除後，重新導向回當前課程頁面並刷新學生清單
}
