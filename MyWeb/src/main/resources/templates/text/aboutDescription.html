使用 WebMvcConfigurer 映射 /about 至 Thymeleaf 視圖

透過在 MyWebConfig 類別中實作 WebMvcConfigurer 並覆寫 addViewControllers() 方法， 可將 /about 等路徑直接對應至視圖（view），無需經過 Controller 類別，適用於無業務邏輯的頁面；這些頁面仍可搭配 Thymeleaf 模板進行動態渲染。 

@Configuration
public class MyWebConfig implements WebMvcConfigurer {

    /**
     * 將 URL 路徑 /about 映射至 Thymeleaf 視圖 nav/about.html，
     * 無需透過 Controller 類別處理。
     */
    @Override
    public void addViewControllers(ViewControllerRegistry registry) {
        registry.addViewController("/about")
                .setViewName("nav/about"); // 對應 src/main/resources/templates/nav/about.html
    }
}
