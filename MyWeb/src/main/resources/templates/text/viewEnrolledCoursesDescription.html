學生已選課程查詢功能

當學生登入系統並進入課程頁面時，系統會從 HttpSession 中取得先前在 Dashboard 中儲存的 "loggedInPerson" 物件，代表當前登入的學生。 該 Person 物件將透過 Model 傳遞至前端，供 Thymeleaf 頁面進行資料綁定與渲染使用。 頁面將顯示學生姓名（person.name），並透過其 Set<Courses> courses 屬性列出所有已註冊的課程，包含課程名稱與費用等資訊。 此功能可協助學生快速查閱其目前所選修的所有課程，有效提升系統的便利性與整體使用體驗。 

@GetMapping("/viewEnrolledCourses")
public String viewEnrolledCourses(HttpSession session, Model model) {
    Person person = (Person) session.getAttribute("loggedInPerson"); // 從 Session 取得當前登入的 Person 物件
    model.addAttribute("person", person); // 將 Person 物件加入 Model，供前端透過 person.name、person.courses 等屬性渲染
    return "authenticated/studentOnly/viewEnrolledCourses"; // 返回學生專屬的課程查詢頁面
}
