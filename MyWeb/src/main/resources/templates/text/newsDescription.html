Spring JDBC 擷取資料 + Thymeleaf 動態渲染

/news 路徑由 NewsController 控制，透過 Spring JDBC API（使用 RowMapper， 非 ORM）呼叫 newsRepository.returnAListOfAllNewsItems() 擷取 List<News>，並將結果傳遞至視圖，最後由 Thymeleaf 進行畫面渲染。

public List<News> returnAListOfAllNewsItems() {
    String sql = "SELECT * FROM news";

    // BeanPropertyRowMapper 透過 setter 方法自動將欄位對應至 News 類別屬性
    return jdbcTemplate.query(sql, BeanPropertyRowMapper.newInstance(News.class));
}
